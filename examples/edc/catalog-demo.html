EDC Catalog Demo

Displaying assets from France TV Access connector

Demo Configuration

This demo fetches the catalog from France TV Access connector:

  • Consumer Connector: consumer.connector-dev.startinblox.com
  • Provider Connector: france-tv-access.connector.luminvent.com
  • Protocol: Dataspace Protocol HTTP (EDC v3 Management API)

This is equivalent to the curl command:

curl -X POST "https://consumer.connector-dev.startinblox.com/management/v3/catalog/request" \
    -H 'Content-Type: application/json' \
    -d '{
  "@context": { "@vocab": "https://w3id.org/edc/v0.0.1/ns/" },
  "counterPartyAddress": "https://france-tv-access.connector.luminvent.com/protocol",
  "protocol": "dataspace-protocol-http"
}'
